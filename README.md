# Nexus Web Studio - Portfolio

A premium, modern, and friendly web studio portfolio built with React 19, Tailwind CSS v4, and Framer Motion. This project is a rebranding work from an elite global agency to a grounded, accessible boutique studio.

## ✨ Project Highlights

- **Rebranded Identity**: Transitioned from high-hype jargon to a trusted, friendly, and quality-driven messaging.
- **Tech Stack**:
  - **Frontend**: React 19 + Vite.
  - **Styling**: Tailwind CSS v4 (Modern utility-first framework).
  - **Animations**: Framer Motion (Glassmorphism, 3D Tilt, Neon Glows).
  - **Icons**: Lucide React.
  - **Data Fetching**: TanStack Query (React Query).
  - **Forms**: React Hook Form + Zod validation.

## 🚀 Key Features

- **Functional Contact Form**: Integrated with **Formspree** for direct-to-inbox submissions.
- **Modern UI/UX**: Magnetic buttons, holographic cards, and fluid 3D parallax effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Premium Performance**: Optimized for fast Lighthouse scores (FCP 0.8s).

## 📂 Project Structure

- `frontend/`: The main React application.
- `mockup-sandbox/`: An isolated playground for UI prototyping and component testing.
- `backend/`: Future placeholder for server-side logic (currently frontend-only).
- `shared/`: Utilities and types shared across the workspace.

## 🛠️ Setup & Development

To run the project locally:

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```
